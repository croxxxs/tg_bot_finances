# 💰 Finance Tracker Bot (aiogram 3)

Асинхронный Telegram-бот для учёта личных финансов, контроля доходов/расходов и отслеживания финансовых целей. Написан на Python с использованием библиотеки `aiogram` и базы данных `SQLite`.

## ✨ Функционал

* **Регистрация пользователей:** Уникальные профили по Telegram ID.
* **Учёт операций:** Добавление доходов (`/add_income`) и расходов (`/add_expense`) с привязкой к категориям.
* **Управление целями:** Установка финансовых целей (`/set_goal`) и автоматическое уведомление при их достижении на основе текущего баланса.
* **Просмотр истории:** Вывод списка транзакций (`/view_transactions`) с фильтрацией по периодам (день, неделя, месяц, год) и категориям.
* **Аналитика:** Статистика (`/statistics`) за выбранный период с процентным распределением расходов по категориям.
* **Удобный интерфейс:** Встроенная клавиатура быстрого доступа (Reply Keyboard).

## 🚀 Быстрый запуск

### 1. Клонирование репозитория и подготовка
Перейдите в директорию с файлом `main.py` (или склонируйте проект):
```bash
cd path/to/your/project
```

### 2.Установка зависимостей
Рекомендуется использовать виртуальное окружение:

```bash
# Создание виртуального окружения
python -m venv venv

# Активация окружения:
# На Linux/macOS:
source venv/bin/activate
# На Windows (CMD):
venv\Scripts\activate
# На Windows (PowerShell):
.\venv\Scripts\Activate.ps1

# Установка необходимых библиотек
pip install aiogram aiosqlite
```

### 3.Настройка переменных окружения

Бот считывает конфиденциальные данные и пути из переменных окружения.

    TG_BOT_TOKEN: Токен вашего Telegram-бота.

    DB_PATH: Путь к файлу базы данных SQLite (необязательно, по умолчанию finance.db).

Передача переменных при запуске:

Для Linux/macOS:
```Bash

export TG_BOT_TOKEN="ВАШ_ТОКЕН_БОТА"
export DB_PATH="finance.db"
```

Для Windows (PowerShell):
PowerShell

```bash
$env:TG_BOT_TOKEN="ВАШ_ТОКЕН_БОТА"
$env:DB_PATH="finance.db"
```

⚠️ Важно: В текущей версии main.py для токена используется встроенное дефолтное значение, но хорошей практикой является передача токена именно через TG_BOT_TOKEN во избежание утечки данных.

### 4.Запуск бота

После настройки окружения запустите скрипт:

```Bash
python main.py
```

При первом запуске бот автоматически создаст файл базы данных и необходимые таблицы (users, transactions, goals).

📘 Доступные команды

Вы можете управлять ботом как через встроенную Reply-клавиатуру, так и текстовыми командами:

    /start — Приветственное сообщение и инициализация клавиатуры.

    /help — Справка по командам.

    /register — Регистрация в системе.

    /login — Проверка статуса авторизации.

    /add_income [сумма] [категория] — Добавить доход (Пример: /add_income 50000 зарплата).

    /add_expense [сумма] [категория] — Добавить расход (Пример: /add_expense 1200 еда).

    /set_goal [сумма] [описание] — Установить финансовую цель (Пример: /set_goal 100000 отпуск).

    /view_transactions [период] [категория] — История транзакций (Доступные периоды: день, неделя, месяц, год).

    /statistics [период] — Просмотр финансового итога и распределения расходов.

🗄 Структура Базы Данных (SQLite)

Проект использует три основные таблицы:

    users — Данные о зарегистрированных пользователях (ID, username, дата).

    transactions — История всех финансовых операций (тип, сумма, категория, дата).

    goals — Финансовые цели пользователей с отметкой о выполнении (notified).
